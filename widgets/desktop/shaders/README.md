## Included Shaders

The included shaders have been modified to work on Qt. Some have also had additional changes, for a full list of changes, open the respective .frag file and check the comment near the top.

Here is a list of included shaders and their sources:

- Space: [https://www.shadertoy.com/view/sldGDf](https://www.shadertoy.com/view/sldGDf)
- Universe: [https://www.shadertoy.com/view/lscczl](https://www.shadertoy.com/view/lscczl)
